Démonstration des sélecteurs CSS

Sélecteur de type

Exemple : p { color: blue; }

Ce texte est affecté par un sélecteur de type (p).

Sélecteur de classe

Exemple : .important { font-weight: bold; }

Ce paragraphe est en gras grâce à une classe.

Sélecteur d'ID

Exemple : #unique { background-color: yellow; }

Ce paragraphe a un ID unique et un fond jaune.

Sélecteurs combinés

Exemple : article p { color: green; }

Ce paragraphe dans un article est vert.

Sélecteurs adjacents

Exemple : h2 + p { font-style: italic; }

Titre

Ce paragraphe suit directement un h2 et est en italique.

Sélecteurs d'attributs

Exemple : input[type="text"] { border: 2px solid red; }

Pseudo-sélecteurs

Exemple : a:hover { color: red; }
