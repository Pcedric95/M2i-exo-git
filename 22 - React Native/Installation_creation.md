# Equivalence HTML dans REACT Native

HTML/WEB    |   React Native

<div>       ->  <View>

<p>         ->  <Text>

<img>       ->  <Image>

<input>     ->  <TextInput>

<form>      ->  <Form>

<button>    ->  <Button>
<button>    ->  <TouchableOpacity> ( <Pressable>) + <Text>


<a>         ->  <Link>
<a href=""> ->  <Navigation>

<select>    ->  <Picker>

<option>    ->  <Picker.Item

<ul>,<li>   ->  <FlatList>

<table>     ->  <Table>

<thead>      ->  <Table.Header>

<tbody>    ->  <Table.Body>

<tr>        ->  <Table.Row>

<td>        ->  <Table.Cell>
